This is my first personal Telegram bot built with Node.js.

It responds to the /start command and sends a simple welcome message.

## Technologies
- Node.js
- node-telegram-bot-api

## How to run
- Install Node.js
- Install dependencies with npm
- Add your bot token in a .env file
- Run the bot with node

## Note
If you face connection or polling errors while running locally,
it is usually related to network or VPN limitations, not the code itself.
